Docstrings


  • Give details about a module or function
  • Should be in every function you write
  • Always inclosed in triple quotes
  • First line summarizes the function, class or module and its purpose
  • Modules: List important classes, functions, excepts
  • Classes: list import methods and/or parts
  • Functions:
    • List and explain parameters one per line
    • If there's a return value, list value; otherwise omit
    • If the function raises exceptions, mention those
  • More details at PEP 257

# Create function with docstring

def docStringExample(arg1, arg2=None):

"""docStringExample(ag1, arg2=None) --> Doesn't do anything, just prints

arg1: the first argument. Whatever you feel like passing

arg2: 2nd argument, defaults to none

"""

print(arg1, arg2)


# Call the docstring in code

print(docStringExample.__doc__)

Variable Arguments List

  • Define functions that can take variable number of parameters
  • `def addition(*numbers)
    • The * denotes it as a variable argument
  • `def log_message(msgType, msg, *params)

r1 = addition(5, 10, 20)

r2 = addition(1, 2, 3)


print("The value of r1 is: ", r1)

print("The value of r2 is: ", r2)

# You can pass a list as args as well

myNums = [5, 10, 20, 1, 2, 3]

# The list requires * to properly enumerate in function

print("The value of list myNums = ", addition(*myNums))



def addition(*args):

# Args is not required, but it is normal convetions

result = 0

for arg in args:

result += arg

return result

Lambda Functions

  • Small, anonymous functions
  • Can be passed as arguments where you need a function
  • Typically used in place just when needed
    • AKA, defining an entirely new function would be adding pointless code
  • Defined as:
    • lambda (parameters) : (expression)
      • Parameters being any arguments
      • Expression being the code block
  • Example without lambda

ctemps = [0, 12, 34, 100]

ftemps = [32, 65, 100, 212]

def CelsiusToFarenheit(temp):

return (temp * 9/5 ) + 32



def FarenheitToCelsius(temp):

return (temp - 32) * 5/9


print(list(map(FarenheitToCelsius, ftemps)))

print(list(map(CelsiusToFarenheit, ctemps)))


  • Or the same code with lambda

print(list(map(lambda t: (t - 32) * 5/9, ftemps)))


print(list(map(lambda t: (t * 9/5) + 32, ctemps)))
